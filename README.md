# Rapid Software Solutions Enterprise Security Program

> **Enterprise Information Security Program aligned with NIST CSF, ISO/IEC 27001, CISSP, GDPR, HIPAA, PCI DSS, and Zero Trust Architecture.**

---

## Overview

This repository contains an enterprise cybersecurity architecture and governance project developed as part of my **Master of Science in Cybersecurity**.

The project presents a comprehensive Enterprise Information Security Program for **Rapid Software Solutions (RSS)**, a fictional multinational organization operating across France, Germany, and Israel. It demonstrates how security governance, risk management, identity and access management, compliance, and secure software development can be integrated into a modern enterprise security program.

## Project Objectives

- Design an enterprise-wide cybersecurity program.
- Align security controls with the NIST Cybersecurity Framework (CSF).
- Address all eight CISSP security domains.
- Support compliance with GDPR, HIPAA, PCI DSS, ISO/IEC 27001, and regional privacy regulations.
- Implement Zero Trust and Defense-in-Depth security principles.
- Develop policies for risk management, incident response, identity management, and secure software development.
